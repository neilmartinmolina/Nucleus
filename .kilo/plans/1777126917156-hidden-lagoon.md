# Nucleus Dashboard Modernization Plan

## Overview
Transform the current dashboard into a modern, structured interface with side navigation panel, DataTables integration with Tailwind CSS, and complete Bootstrap-to-Tailwind migration.

## Key Changes Required

### 1. Structured Dashboard Layout
- **Side Panel Navigation**: Fixed sidebar with user info and navigation links
- **Main Content Area**: Dynamic content loading based on URL params
- **Responsive Design**: Mobile-friendly collapsible sidebar
- **Consistent Navigation**: Replace all page navigation with single-page application approach

### 2. URL Parameter Navigation
- **Parameter Pattern**: `?tab=usermanagement`, `?tab=websites`, `?tab=folders`, etc.
- **Tab System**: JavaScript-based tab switching without page reloads
- **State Management**: Maintain tab state across navigation
- **Bookmark Support**: URL-friendly parameters for direct access

### 3. DataTables Integration (Tailwind Version)
- **Library**: Use DataTables with Tailwind CSS styling
- **Features**: Search, sort, pagination, responsive design
- **Data Sources**: Replace current PHP table generation with JavaScript DataTables
- **Server-side Processing**: AJAX calls for large datasets
- **Styling**: Custom Tailwind CSS integration for DataTables

### 4. Complete Bootstrap to Tailwind Migration
- **Files to Convert**: dashboard.php, users.php, folders.php, view-folder.php, update.php
- **Components**: Cards, tables, forms, buttons, modals, navigation
- **Utility Classes**: Replace Bootstrap classes with equivalent Tailwind utilities
- **Interactive Elements**: Replace Bootstrap JavaScript with Tailwind equivalents

### 5. Page Structure Changes

#### Dashboard Components
- **User Summary**: User info card with role, last login
- **Quick Stats**: Project counts, update statistics
- **Recent Activity**: Real-time updates and notifications
- **Action Buttons**: Quick access to common tasks

#### User Management Tab
- **Enhanced Users Table**: DataTables with search/filter
- **User Actions**: Edit, delete, permissions management
- **Role Management**: Visual role indicators and bulk actions
- **User Creation**: Inline form with validation

#### Website Management Tab
- **Project Overview**: All websites with status indicators
- **Update Interface**: Simplified update process
- **Version Control**: Visual version tracking
- **Status Management**: Quick status updates

#### Folder Management Tab
- **Folder Browser**: Hierarchical folder display
- **Project Organization**: Drag-and-drop project assignment
- **Permission Controls**: Folder-level access management
- **Folder Actions**: Create, rename, delete with confirmation

#### Navigation Logic
```javascript
// URL-based tab switching
function switchTab(tab) {
    window.history.pushState({ tab }, '', `?tab=${tab}`);
    loadTabContent(tab);
}

// Tab content loading
function loadTabContent(tab) {
    fetch(`index.php?tab=${tab}`)
        .then(response => response.text())
        .then(html => {
            document.getElementById('tab-content').innerHTML = html;
            initializeDataTables();
        });
}
```

#### DataTables Integration
```javascript
// Initialize DataTables with Tailwind styling
$('#websites-table').DataTable({
```
